Labels

Everything denominated in years

Labelling points

Value Dimension Length Rule Example
Start day Time \(\ge 2\) Year-Month-Day 2000, 2000.003 \(\to\) "2000-01-01", "2000-01-02"
Start month Time \(\ge 2\) Year-Month 2000, 2000.085 \(\to\) "2000-01-01", "2000-02-01
Start quarter Time \(\ge 2\) Year-Quarter 2000, 2000.249 \(\to\) "2000-01-01", "2000-04-01"
Any Any Any Value 80.1 \(\to\) "80.1"

If every point is the start of a day, and more than one point, then label using date, If every point is the start of a month, and more than one point, then label using date. Eg 2029.937, 2030.022, 2030.101, 2030.186 is displayed as 2000-01-01, 2000-02-01, 2000-03-01, 2000-04-01.

Otherwise, label with number.

Labelling intervals

Lower limit Upper limit Dimension Rule Example
Integer Lower+1 Age Lower 5, 6 \(\to\) "5"
Integer Lower+1 Time Lower or Upper 5, 6 \(\to\) "5" or "6"
Integer Integer Age Lower-<Upper-1> 5, 10 \(\to\) "5-9"
Start day Start next day Time Year-Month-Day 2000, 2000.003 \(\to\) "2000-01-01"
Start month Start next month Time Year-Month 2000, 2000.085 \(\to\) "2000-01"
Start quarter Start next quarter Time Year-Quarter 2000, 2000.249 \(\to\) "2000-Q1"
Any Infinite Any Lower+ 100, Inf \(\to\) "100+"
Infinite Any Any <Upper -Inf, 2000 \(\to\) "<2000"
Any Any Age or Time Lower-Upper 77, 80.1 \(\to\) "77-80.1"

Whether a one-year time interval is displayed using lower or upper depends on labelStart argument.

Only use “2000-01” labelling if every interval can be interpreted as a month and if months successively. Similarly, only use “2000-Q1” labelling if every interval can be interpreted as a quarter.
